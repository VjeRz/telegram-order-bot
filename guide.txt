1. HOW TO REGISTER
================================================================================
All first‑time users must register.

Step by step:
1. Send /register to the bot.
2. Enter your full name.
3. Enter your email address.
4. Select your role group (Agency, Branch, or Technician).
5. Select your sub‑role:
   - Agency: Team Leader, Salesforce
   - Branch: Inputters, Supervisor, Manager, IT
   - Technician: Technician, HSA
6. If you are from Agency, also choose your WOK (Working Area) and enter your SF ID.
7. After registration, your status is "pending". IT will approve you.
8. You will receive a message when approved. Then you can use /start.

Note: Only the IT role can approve registrations. IT users see pending registrations via /pending.

3. ORDER LOOKUP (/start)
================================================================================
After approval, send /start. The bot will reply:
"Semangat Pagi, Masukan Order ID
Contoh: AOs326032509275620607db90"
Type or paste the Order ID. The bot will reply with:
- Order ID
- STO
- WOK
- Order Status
- Channel Name
- Fallout Reason
- SalesForce
- Tanggal Complete
- Tanggal Input
- Sub Error Code
- Technician Notes

If the Order ID is not found, you will be told to try again.

4. SALES PERFORMANCE REPORT (/salesreport)
================================================================================
Available for: Supervisor, Team Leader, IT, Manager.

How to use:
1. Send /salesreport.
2. Choose a WOK:
   • MANADO TALAUD
   • BOLAANG MONGONDOW
   • GORONTALO - PAHUWATO
   • BITUNG MINAHASA
3. Choose a Channel:
   • B2B2C&OTHERS
   • AGENCY
   • GRAPARI
   • SOBI AFFILIATE
   • WEB&APP
   (Team Leader: channel is forced to AGENCY – no choice.)
4. Choose a Month (January to December).
5. The bot processes data and sends the report.

What the report shows:
- For AGENCY channel:
   – Each salesperson’s name, total orders, and counts per status (all 27 statuses + OTHER).
   – Team Leader also sees a list of Order IDs with their status.
- For other channels (GRAPARI, etc.):
   – Aggregated totals per status for that channel and WOK.

The report is split into multiple messages (one per salesperson) to avoid character limits.

================================================================================
5. USAGE REPORT (/report)
================================================================================
Available for: Manager, Supervisor, HSA, IT.

Shows how many times each registered user has used /start to check orders.

Commands:
/report day        – Today’s usage (text summary)
/report week       – This week’s usage (text)
/report month      – This month’s usage (CSV file)
/report 2026-04-01 2026-04-10 – Custom date range (CSV file)

The CSV file can be opened in Excel or Google Sheets.

8. TROUBLESHOOTING
================================================================================
Bot doesn't respond /start:
- You are not registered or not approved. Register with /register and wait for IT approval.
- The bot may be down. Send /ping – if it replies "pong", the bot is alive.

Order ID not found:
- Check for typos, extra spaces, or wrong case. The bot ignores case but not spaces.
- The Order ID may not exist in the order sheet.

Sales report empty or error:
- Ensure you selected a WOK, Channel, and Month that have data.
- If you see "Message_too_long", it has been fixed – contact IT if still occurs.

Approval buttons not appearing:
- Only the IT user (Telegram ID set in environment variable) receives them. If you are IT, check your chat with the bot.

Need to change role or re‑register?
- Contact IT. They can manually edit the Users sheet or delete your entry so you can re‑register.
