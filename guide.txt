================================================================================
                                USER GUIDE
================================================================================

HOW TO START
------------
1. Open Telegram and search for the bot.
2. Type /start or click “Start”.
3. If you are not registered, the bot will ask you to register.

REGISTRATION
------------
1. Click the “Daftar” (Register) button.
2. Enter your full name.
3. Enter your email address.
4. Choose your role group (Agency, Branch, Technician, or Grapari).
5. Choose your subrole.
6. If you are from Agency, also select your Working Area (WOK) and enter your Salesforce ID.
7. Your request is sent to IT for approval. You will receive a message when approved.

MAIN MENU BUTTONS (after approval)
-----------------------------------
- 🔍 Cek Order         → Check a single Order ID.
- 📦 Cek Banyak Order  → Check 2‑10 Order IDs at once (separate by spaces, commas, or new lines).
- 📊 Cek Laporan Sales → Sales reports (only for certain roles).
- 📊 Laporan Pengguna Bot → Usage report (only for Managers, Supervisors, HSA, IT).
- 📖 Panduan Pengguna  → Role‑based help text.

ORDERS (SINGLE & BULK)
----------------------
- Single order: enter one Order ID. The bot shows all available fields.
- Bulk order: enter 2‑10 Order IDs separated by spaces, commas, or new lines. The bot shows each found order and lists any missing IDs.

SALES REPORTS – WHAT EACH ROLE CAN SEE
---------------------------------------

A. Manager / Supervisor / IT
   - Laporan Detail (per WOK):
       Choose WOK → Year → Month → see a breakdown of order statuses (including percentages) for that WOK.
   - Ringkasan (per WOK & Channel):
       Choose Year → Month → bot sends 11 tables:
         1) Per WOK summary
         2) Global per‑channel summary
         3‑6) Per‑WOK channel summaries
         7) Global popular paket (all completed packages)
         8‑11) Per‑WOK popular paket
   - Laporan Pengguna Bot:
       Choose Daily, Weekly, or Monthly to see who used the bot and how often. Monthly reports are sent as a CSV file.

B. Team Leader (Agency)
   - Laporan Detail (per WOK):
       Choose WOK → Year → Month → then choose:
         * 📥 Download CSV   → all orders (any status) as a CSV file.
         * 📋 Tampilkan Data → per‑salesperson list with order details.
         * 📦 Popular Paket (Agency) → list of completed packages (Agency only) with counts and percentages.
   - (No summary report, no usage report.)

C. Team Leader Grapari / CS Grapari
   - Same as Team Leader (Agency), but all data is filtered to GRAPARI channel.
   - Additional report: “Grapari Performance (per STO)”:
       Choose Year (2025 or 2026) → Month → then choose:
         * 📊 Lihat Ringkasan → summary table by STO (city/area) (Input, Completed, Fallout, IO/PS, Contribution, MoM).
         * 📥 Download CSV → all orders (any status) for that month as a CSV file.

D. Inputters, Salesforce, Technician, HSA
   - Only order lookup (single / bulk). No sales reports.

USAGE REPORT (for Managers, Supervisors, HSA, IT)
-------------------------------------------------
- Click “Laporan Pengguna Bot” or type /report followed by day|week|month.
- Daily / weekly: text summary of who searched, how many times, and duration.
- Monthly: CSV file with detailed columns (TelegramID, Name, SubRole, number of lookups, first lookup, last lookup, duration).

GUIDE COMMAND
-------------
- Type /guide or /help anywhere. The bot shows a help page tailored to your role.

CANCELLING AN OPERATION
-----------------------
- Type /cancel at any time to abort the current conversation and return to the main menu.

TROUBLESHOOTING
---------------
- If the bot does not respond, try /start again.
- If an Order ID is not found, the bot shows the last update date (with time) and asks you to try another ID.
- If you see an error, please contact IT with a screenshot of the error message.
